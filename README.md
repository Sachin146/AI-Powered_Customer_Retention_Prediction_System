<h2>📌 Project Overview</h2>

<p>
Customer churn prediction is one of the most important business problems in the telecom industry.
This project uses Machine Learning techniques to predict whether a telecom customer is likely to leave the service or stay.
</p>

<p>
The system includes:
</p>

<ul>
<li>✅ Data Preprocessing Pipeline</li>
<li>✅ Feature Engineering</li>
<li>✅ Missing Value Handling</li>
<li>✅ Feature Scaling</li>
<li>✅ Flask Web Application</li>
<li>✅ ML Model Prediction</li>
<li>✅ Beautiful User Interface</li>
<li>✅ Deployment Ready Structure</li>
</ul>

<h2>🚀 Installation & Setup</h2>

<h3>1️⃣ Clone Repository</h3>

<pre>
git clone https://github.com/your-username/customer-retention-prediction-system.git
</pre>

<h3>2️⃣ Navigate to Project</h3>

<pre>
cd customer-retention-prediction-system
</pre>

<h3>3️⃣ Create Virtual Environment</h3>

<pre>
python -m venv .venv
</pre>

<h3>4️⃣ Activate Environment</h3>

<pre>
.venv\Scripts\activate
</pre>

<h3>5️⃣ Install Dependencies</h3>

<pre>
pip install -r requirements.txt
</pre>

<h3>6️⃣ Run Flask Application</h3>

<pre>
python app.py
</pre>
